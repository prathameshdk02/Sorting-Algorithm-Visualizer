How to Use?

- Select any Algorithm from the Sidebar....

- A Randomly generated Array would show up in visualizer's display...

- Hover over the "Play" button to see a list of functions supported....

- Keep an eye on the Title - Be Sure you have already selected an Algorithm...

About Buttons

Play - To Start Visualization

Stop - To Stop Visualization

Reset - To Generate a Random Array

Slow Down - To Slow down the animating speed of Visualizer

Fast Forward - To Speed up the animating speed of Visualizer

Help - To bring up this Dialogue Box anytime...

Have a Happy Time Visualizing.....
